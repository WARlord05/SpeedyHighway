# Analytics

Data schema validation and summary statistics for SpeedyHighway game data.

- **schema.py** – Validate `game_data.json` structure.
- **summary_stats.py** – Compute games played, playtime, achievements, score stats.
- **fixtures/game_data_sample.json** – Sample data for testing and analysis.

See **docs/DATA_ANALYSIS.md** for the full data dictionary and how to run the analysis script.
